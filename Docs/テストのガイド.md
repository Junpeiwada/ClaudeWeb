# テストのガイド

## テスト修正の禁止（確認必須）

テストが失敗した場合、以下の手順を厳守すること。

1. **テストコードのEdit/Writeは絶対に行わない**（ユーザー承認があるまで）
2. 失敗原因を分析し、**実装コードの修正で対応できないか**を先に検討する
3. テスト側の変更が必要と判断した場合、以下をユーザーに提示して**承認を待つ**：
   - 失敗の原因
   - 実装側では対応できない理由
   - テストの具体的な修正内容
4. ユーザーが明示的に承認した場合のみ、テストコードを修正する

## テスト環境

- **フレームワーク**: Playwright（E2E、Chromiumのみ）
- **テストディレクトリ**: `tests/`
- **設定ファイル**: `playwright.config.ts`
- **フロントエンド**: Vite dev server（ポート5173）が自動起動

## テスト実行コマンド

```bash
# 全テスト実行
npm test

# UI モード
npm run テスト:UIモード

# 単一テスト実行
npx playwright test tests/chat.spec.ts

# 特定テスト名で実行
npx playwright test -g "テスト名"
```

## テストの書き方

### SSEモックパターン

サーバーのSSEレスポンスは `page.route()` でモックする。

```typescript
function sseBody(events: object[]): string {
  return events.map((e) => `data: ${JSON.stringify(e)}\n\n`).join("");
}

await page.route("/api/chat", (route) => {
  return route.fulfill({
    status: 200,
    headers: {
      "Content-Type": "text/event-stream",
      "Cache-Control": "no-cache",
    },
    body: sseBody([
      { type: "session_id", sessionId: "..." },
      { type: "text", content: "Hello" },
      { type: "done", sessionId: "..." },
    ]),
  });
});
```

### SSEイベントタイプ

| タイプ | 用途 |
|--------|------|
| `session_id` | セッションID通知 |
| `text` | アシスタントのテキスト（チャンク単位） |
| `activity` | ツール実行中の表示 |
| `tool_result` | ツール結果 |
| `permission` | 権限要求 |
| `limit_error` | レート制限エラー |
| `error` | 汎用エラー |
| `done` | ストリーム完了 |
| `reconnect_state` | 再接続時の状態復元 |

### Loading状態のシミュレーション

リクエストをハングさせることで、フロントエンドをloading状態に保つ。

```typescript
await page.route("/api/chat", async () => {
  await new Promise(() => {}); // 応答しない → fetchがペンディング
});
```

### ネットワークエラーのシミュレーション

```typescript
route.abort("connectionreset");   // 接続リセット
route.abort("connectionrefused"); // 接続拒否
```

### 共通ヘルパーパターン

各テストファイルで共通のヘルパーを定義する（共有ファイルは使わない）。

```typescript
const TEST_REPOS = [
  { id: "TestRepo", name: "TestRepo", path: "/tmp/TestRepo" },
];

async function mockRepos(page: Page) {
  await page.route("/api/repos", (route) =>
    route.fulfill({ json: TEST_REPOS })
  );
}

async function selectRepo(page: Page) {
  await page.getByText("Select repo").click();
  await page.getByText("TestRepo").nth(0).click();
}
```
