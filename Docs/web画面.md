### ヘッダー

```
┌──────────────────────────────────────┐
│ ClaudeWeb              [blog ▼] [+]  │
└──────────────────────────────────────┘
  ↑ タイトル              ↑ リポジトリ  ↑ 新規セッション
                           選択          ボタン
```

- **リポジトリ選択** `[blog ▼]`: ドロップダウン。選択するとセッションが切り替わる
- **新規セッション** `[+]`: 現在のリポジトリで新しい会話を開始する

### メッセージ表示

メッセージは吹き出し形式で左右に分ける。

- **Claude（左寄せ）**: 背景色で区別（薄いグレー等）
- **ユーザー（右寄せ）**: 背景色で区別（薄い青等）
- Claude の応答は **Markdown レンダリング**（コードブロック、リスト、太字等）
- ストリーミング中はテキストが逐次追加され、末尾にカーソル点滅アニメーション

### 入力エリア

```
┌──────────────────────────────────────┐
│ [メッセージを入力...        ] [送信]  │
└──────────────────────────────────────┘
```

- テキストエリア（複数行対応、改行は Shift+Enter）
- Enter キーで送信（モバイルではソフトキーボードの改行で送信）
- 送信ボタンは処理中にはグレーアウト
- Claude 処理中は入力欄の下に「処理中...」のインジケータを表示

## 処理中の表示

Claude Code が応答を生成している間の表示。

```
│  ┌─────────────────────────────────┐ │
│  │ 🤖 ファイルを確認しています...  │ │
│  │    ████░░░░░░                   │ │
│  │    ← テキストが逐次追加される    │ │
│  └─────────────────────────────────┘ │
│                                      │
├──────────────────────────────────────┤
│ [入力不可（処理中...）       ] [---]  │
└──────────────────────────────────────┘
```

- メッセージ入力は無効化（1セッション・1リクエストずつ処理のため）
- 送信ボタンは無効化
- Claude の吹き出し内にストリーミングテキストが流れる

## 権限承認 UI

Claude Code がツール使用の許可を求めたとき、チャット内にインライン表示する。

### 承認待ち状態

```
│  ┌─────────────────────────────────┐ │
│  │ 🤖 記事ファイルを作成します。    │ │
│  └─────────────────────────────────┘ │
│                                      │
│  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐ │
│  │ ⚠️  権限の確認                  │ │
│  │                                 │ │
│  │ write: ファイル書き込み          │ │
│  │ posts/2026-03-25-fishing.md     │ │
│  │                                 │ │
│  │    [許可する]    [拒否する]      │ │
│  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘ │
│                                      │
├──────────────────────────────────────┤
│ [入力不可（承認待ち...）     ] [---]  │
└──────────────────────────────────────┘
```

- 目立つカード（ボーダー付き、黄色/オレンジ系の背景）
- ツール名と操作内容を表示
- **[許可する]**: 緑色ボタン。押すと処理が続行
- **[拒否する]**: 赤色ボタン。押すと Claude にツール使用が拒否されたことを通知
- 承認待ちの間、メッセージ入力は無効化

### 承認後の表示

```
│  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐ │
│  │ ✅ 許可しました                  │ │
│  │ write: posts/2026-03-25-...     │ │
│  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘ │
```

- 承認/拒否後はコンパクトな表示に折りたたむ（ログとして残す）
- 承認: 緑、拒否: 赤 のアイコンで区別

## 再接続時の表示

iPhone スリープ等で接続が切れた後、再度アクセスした場合。

```
┌──────────────────────────────────────┐
│ ClaudeWeb              [blog ▼] [+]  │
├──────────────────────────────────────┤
│                                      │
│        🔄 セッションに再接続中...     │
│                                      │
└──────────────────────────────────────┘
```

`GET /api/status` でセッション状態を取得し：

- **処理中セッションあり** → チャット履歴を復元し、ストリーミングを再開
- **承認待ちセッションあり** → チャット履歴を復元し、権限承認 UI を再表示
- **アクティブセッションなし** → 通常のチャット画面を表示

## レスポンシブ対応

### iPhone (375px〜)

- ヘッダー: タイトルを省略し `[リポジトリ ▼] [+]` のみ
- メッセージ吹き出し: 画面幅いっぱい（左右マージン小）
- 入力欄: 画面下部に固定（position: sticky）
- 権限承認カード: 画面幅いっぱいに表示

```
┌─────────────────────┐
│ [blog ▼]        [+]  │
├─────────────────────┤
│                      │
│ 🤖 どのような作業   │
│ をしますか？         │
│                      │
│    写真で記事書いて 👤│
│                      │
│ ⚠️ 権限の確認        │
│ write: posts/...     │
│ [許可する] [拒否する] │
│                      │
├─────────────────────┤
│ [入力...      ][送信]│
└─────────────────────┘
```

### タブレット / PC (768px〜)

- チャット領域の最大幅を制限（800px 程度、中央寄せ）
- 吹き出しの幅に余裕を持たせる
